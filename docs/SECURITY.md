# Security Policy / Política de Segurança

**PT-BR** | [EN](#english)

---

## PT-BR

### Relatando uma vulnerabilidade

Se você descobrir uma vulnerabilidade de segurança no vidgrab, **não a divulgue publicamente**. Em vez disso:

1. **Envie um e-mail privado** para [gustavocode18@gmail.com](mailto:gustavocode18@gmail.com) com:
   - Descrição da vulnerabilidade
   - Passos para reproduzir (se possível)
   - Impacto potencial
   - Versões afetadas

2. Você receberá uma resposta dentro de 48 horas confirmando recebimento

3. Trabalharemos em um patch e agendaremos uma release de segurança

4. Você será creditado na release notes (a menos que prefira permanecer anônimo)

### Versões suportadas

| Versão | Suportada | Até |
| --- | --- | --- |
| 0.4.x | ✅ | Até próxima minor |
| 0.3.x | ✅ | Até 0.5.0 |
| 0.2.x | ⚠️ | Patches críticos apenas |
| < 0.2.0 | ❌ | Não suportado |

## English

### Reporting a vulnerability

If you discover a security vulnerability in vidgrab, **do not disclose it publicly**. Instead:

1. **Send a private email** to [gustavocode18@gmail.com](mailto:gustavocode18@gmail.com) with:
   - Description of the vulnerability
   - Steps to reproduce (if possible)
   - Potential impact
   - Affected versions

2. You will receive a response within 48 hours confirming receipt

3. We will work on a patch and schedule a security release

4. You will be credited in the release notes (unless you prefer to remain anonymous)

### Supported versions

| Version | Supported | Until |
| --- | --- | --- |
| 0.4.x | ✅ | Until next minor |
| 0.3.x | ✅ | Until 0.5.0 |
| 0.2.x | ⚠️ | Critical patches only |
| < 0.2.0 | ❌ | Not supported |
